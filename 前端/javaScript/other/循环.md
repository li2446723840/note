#### 跳出循环(continue、break)

<font color="red">continue</font>关键字，可以跳出这次循环，直接进入下一次循环

```js
for (var i = 0; i < 5; i++) {
       if(i==3){
          continue;
       }
       console.log(i);//0 1 2 4
    }
```

<font color="red">break</font>关键字，可以终止循环，直接让整个循环结束运行

```js
for (var i = 0; i < 5; i++) {
       if(i==3){
          break;
       }
       console.log(i);//0 1 2
    }
```
